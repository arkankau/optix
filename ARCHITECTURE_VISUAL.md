## Summary

**What you built:**
A desktop overlay app that captures your screen, applies real-time GPU-accelerated vision correction using deconvolution shaders, and displays the corrected output in a transparent always-on-top window.

**Like:** Clue.ly (overlay) + OBS (capture/GPU) + Corrective lenses (vision)

**Performance:** 60 FPS at 1080p with <16ms latency

**User Experience:** 
1. Set it and forget it (runs in tray)
2. Adjust on-the-fly (control panel)
3. Toggle anytime (hotkey)
4. Works with everything (transparent overlay)
